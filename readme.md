# Dev teams UnivPortal
A simple coursework website for professors and students.

## Tech-Stack
- PostgreSQL
- Prisma
- GraphQL
- Vue.js
- TailwindCSS

## Commands
- Move to `server` folder and run `npx ts-node src/index.ts`
- Move to `client` folder and run `npm run dev`
